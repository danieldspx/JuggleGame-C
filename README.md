## Install Allegro
Primeiramente precisamos instalar o Allegro.
#### Linux

    sudo add-apt-repository ppa:allegro/5.2
    sudo apt-get update
    sudo apt-get install liballegro5-dev

## How to Run
Entre na pasta do projeto e digite
#### Linux

    ./run.sh

ou



    make && ./bin/main

Pronto, você já pode jogar.
